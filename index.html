参考经典公式化 Alpha 因子
Alpha#12 = sign(Δvolume) × (-1 × Δclose)

量价背离因子。当成交量增加而价格上涨时，因子值为负（可能的看空信号）； 当成交量减少而价格下跌时，因子值为正（可能的看多信号）。

Alpha#101 = (close - open) / ((high - low) + 0.001)

日内价格位置因子。衡量收盘价在当日价格区间中的相对位置。 值接近 +1 表示收盘接近最高价（强势），接近 -1 表示收盘接近最低价（弱势）。

技术指标计算方法与作用

📈 RSI 相对强弱指数

作用：衡量价格变动的速度和幅度，识别超买（>70）和超卖（<30）区域。

计算步骤：

  • 计算价格变化 Δ = close - close(-1)
  • 分别计算上涨和下跌的平均值
  • RS = 平均上涨 / 平均下跌
  • RSI = 100 - 100/(1 + RS)

📊 MACD

作用：判断趋势方向、强度和转折点。DIF 上穿 DEA 为金叉，下穿为死叉。

计算步骤：

  • DIF = EMA(close, fast) - EMA(close, slow)
  • DEA = EMA(DIF, signal)
  • MACD柱 = 2 × (DIF - DEA)

🎯 布林带 Bollinger Bands

作用：衡量价格波动率和相对位置。价格触及上轨可能超买，触及下轨可能超卖。

计算步骤：

  • 中轨 = N日收盘价的简单移动平均
  • 标准差 = N日收盘价的标准差
  • 上轨 = 中轨 + k × 标准差
  • 下轨 = 中轨 - k × 标准差
